# ♟️ 2-Player Chess Game

A browser-based chess game for two players built using HTML, CSS, and JavaScript. The project focuses on implementing core chess rules, turn-based gameplay, and interactive UI in a frontend-only environment.

## 🚀 Features
- Two-player local gameplay (same device)
- Turn-based system
- Legal move validation for all pieces
- Dynamic game state management
- Interactive chessboard UI

## 🛠️ Tech Stack
- HTML
- CSS
- JavaScript

## 📌 Project Status
In progress

## ▶️ How to Run
1. Clone the repository  
2. Open `index.html` in your browser
